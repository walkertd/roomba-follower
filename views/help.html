API Documentation

/api/records/mission/$mission

Return the recorded data for the given mission ($mission).

/api/records/mission/$mission/cycle/$cycle

Return the recorded data for the given mission ($mission) filtered on cycle ($cycle).

/api/records/mission/$mission/phase/$phase

Return the recorded data for the given mission ($mission) filtered on phase ($phase).

/api/records/mission/$mission/cycle/$cycle/phase/$phase

Return the recorded data for the given mission ($mission) filtered on cycle ($cycle) and phase ($phase).

/api/records/id/$id

Return the record matching id = $id.

/api/records/latest

Return the latest record added to the database.

/api/records/all

Return all recordings.

/api/records

Return all recordings.

/api/list/missions

Return a list of mission numbers.
